=================
Helios User Guide
=================

Overview
--------
This guide provides instructions for setting up your repository to work with Helios, especially all the repository-wide settings and environment setup.

Prerequisites
--------------
- GitHub account with maintainer permissions in the repository
- Available GitHub environments for deployment
- Helios GitHub App installed in the repository

Repository Setup
----------------

1. Navigate to the repository settings
   - Go to your repository on GitHub.
   - Click on the **Settings** tab.

2. Configure Locking Thresholds and Java-Package Names

3. Generate Shared Secret 
   - In the **Helios** section, click on **Generate Shared Secret**.
   - Copy the generated secret and store it securely. 

4. Fetch latest Workflows and confingure them

5. Add Test Types
